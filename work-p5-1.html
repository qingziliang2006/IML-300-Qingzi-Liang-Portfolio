p5.js 1

Description

This sketch was created as an early exercise in IML 300 to explore basic shapes, color, and motion in p5.js. It focuses on understanding how simple visual rules can generate a mood and how small changes in parameters affect the overall feel of the scene.

The piece is less about a polished final image and more about learning how to think with code. It represents a starting point for testing composition, layering, and timing in a playful, low-pressure way.
