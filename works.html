Major Projects

Interactive Projects

Three core projects from IML 300.

Project 1 • Interactive Audio & Visual (p5.js)

Real-time audio–visual interaction using p5.js.

This p5.js project explores the relationship between sound, motion, and user interaction. Visual elements respond to rhythm, amplitude, and cursor movement, creating a dynamic space that feels alive and reactive. The goal was not to create a fixed animation but an open-ended audiovisual environment that users shape through presence and gesture.

Project 2 • Dive Into the Deep — HyperNarrative

GitHub Pages • nonlinear storytelling

This hypertext narrative guides users through an imagined underwater world, structured through branching exploration rather than linear storytelling. Users navigate scenes, creatures, and interactive elements through hyperlinks that act as narrative choices. This project experiments with mood, spatiality, and environmental storytelling while demonstrating how HTML-based navigation can become a narrative mechanic.

View HyperNarrative

Project 3 • Networked Care — Community Website

GitHub Pages • community storytelling & care design

Networked Care is a conceptual community website exploring how online platforms can foster reciprocity and emotional support rather than extraction. With a gentle visual system and simple interaction patterns, the project invites visitors to share, reflect, and participate. It also considers how UI choices communicate care, softness, and invitation. The About section properly credits the site as “Created by Zoe Leung.”

Reflection

Vibe Coding Reflection

This semester, vibe coding shifted how I understand both creativity and technical experimentation. Instead of treating each sketch as a finished project, I approached them as atmospheric studies—ways to quickly test feelings, textures, moods, and small systems. The looseness of vibe coding allowed me to explore intuition, randomness, and accidents that I normally avoid when trying to “polish” a work. I also became more aware of repetitive generative patterns and learned to question why certain visuals feel satisfying. Overall, vibe coding expanded my sense of play in creative code and helped me build a more flexible, exploratory practice.
